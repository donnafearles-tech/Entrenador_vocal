# 🎤 Entrenador Vocal con Hume AI

Aplicación web con Streamlit que analiza la entonación, ritmo y tonalidad de tu voz usando la API de Hume AI, y te da recomendaciones para sonar más **persuasiva**, **directa** o **experta**.

---

## 📋 Requisitos previos

- Python 3.8 o superior instalado.
- Una API Key de **Hume AI**. Puedes obtenerla registrándote en [dev.hume.ai](https://dev.hume.ai).
- Un archivo de audio en formato `.wav` o `.mp3` con tu voz (sin música de fondo ni ruido excesivo).

---
